Number of Incidents by Location

SELECT COUNT(location_idlocation) FROM studentalert.incident, studentalert.location WHERE idlocation = location_idlocation GROUP BY location_idlocation

Each row number corresponds to the location id number.

If there is a 5 in row 51, that means 5 incidents happened at the location with id = 51
